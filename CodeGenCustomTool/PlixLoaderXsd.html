PLiXLoader Schema
http://schemas.neumont.edu/CodeGeneration/PLiXLoader

Elements

  • sourceFiles: Settings for files with the CustomTool property set to NUPlixLoader. Provides the schema for the Plix.xml file, which should be placed in the project's root directory.
    • sourceFile (optional, repeat): A file in the project with a CustomTool property set to NUPlixLoader.
      • Choose One (optional, repeat)
        • extensionClass (optional): Add a set of available extensions functions for use by the transform.
          • @xslNamespace (xs:string, required): The namespace for this extension in the transform file.
          • @className (xs:string, required): A fully qualified name for loading the class to instantiate and use with this extension
        • projectReference (optional): Ensure that the project references the specified assembly.
          • @namespace (xs:string, required): The assembly name used to test for existence of the reference in the project with the EnvDTE.References.Item method.
          • @assembly (xs:string, required): The name of the assembly file used to add a reference to the project if it is not already in the references collection.
        • transformParameter (optional): Pass an in parameter to the transform.
          • @name (xs:string, required): The name parameter pass into the transform file. Matches the name attribute of an xsl:param element in the transform.
          • @value (xs:string, required): The value to pass to the transform for this parameter.
      • @file (xs:string, required): The name of the file relative to the location of the plix.xml file. Plix.xml should be placed in the root directory of the project.
      • @transformFile (xs:string): The transform file used to process the source file. If transformFile is not specified, then the default is the specified file with the extension replaced by '.xslt' or '.xsl' extensions.
      • @liveDocumentObject (xs:string): If the document is currently open in an editor and this value is set, then retrieve the file stream from automation model using this name.
